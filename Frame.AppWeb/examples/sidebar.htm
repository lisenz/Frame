SideBar

SideBar 作为侧边栏控件，通过在页面的 div 标签添加 .frame-sidebar进行标识或者直接通过使用 $("#页面标签id").SideBar(options); 这样一行代码，即可将目标元素通过 options 相关属性的配置生成相应的侧边栏控件。
进行标识的侧边栏控件标签，需要放置在 .frameui-container-inner的 div 容器中。详细布局结构参照 代码 标签页中 HTML 示例代码的布局。

Options 属性对象详细说明见下表：
属性名 具体描述 类型 默认值
Shortcuts 侧边栏顶部的快捷菜单。只固定支持配置4个快捷菜单项。 Array null
Content 侧边栏中包裹的内容，这里可以一个function函数返回jquery标签对象或者字符串表示的标签。若侧边栏容器在页面已包含有子元素标签，则优先使用已包含内容标签。 String/Function null
侧边栏的 Shortcuts 快捷项对象的属性详细说明见下表：
属性名 具体描述 类型
Icon 快捷菜单项的显示图标。 String
Mode 快捷菜单项的背景色模式。有8种模式可选：transparent，info，primary，success，warning，default，error，inverse。 String
Title 快捷菜单项的功能描述提示文本。 String
OnClick 快捷菜单项的单击事件。 Function

html代码：

<div class="frameui-container">
    <div class="frameui-container-inner">
        <div id="side" class="frame-sidebar">
            <div style="height:300px; color:#333; padding:5px;">这里可以放置内容</div>
        </div>
        <div class="frameui-container-content for-sidebar">
            右侧内容容器
        </div>
    </div>
</div>

javascript代码：

$("#side").SideBar({
    Shortcuts: [
        { Icon: "icon-signal", Mode: "success", OnClick: function () { $.Messager.Alert("系统消息", "icon-signal", "info"); } },
        { Icon: "icon-pencil", Mode: "info", OnClick: function () { $.Messager.Alert("系统消息", "icon-pencil", "info"); } },
        { Icon: "icon-group", Mode: "warning", OnClick: function () { $.Messager.Alert("系统消息", "icon-group", "info"); } },
        { Icon: "icon-cogs", Mode: "error", OnClick: function () { $.Messager.Alert("系统消息", "icon-cogs", "info"); } }
    ]
});
