BreadCrumbs

BreadCrumbs 是一个面包屑控件，可以嵌入到页面的任何位置。通过在 div 标签添加 .frame-breadcrumbs 标识为 Frame 面包屑控件。
继而使用 var bread = $("#页面标签id").BreadCrumbs(options); 这样一行代码，根据 options 相关属性的配置将目标元素生成为面包屑控件对象。

Options 属性对象详细说明见下表：
属性名 具体描述 类型 默认值
Items 面包屑的项对象集合。 Array null
ToolBar 位于面包屑控件右侧的工具栏。支持设置文本框控件和按钮控件两种类型控件。当设置按钮控件时，options 格式：｛Type，Icon，Desc，OnClick｝，其中Type 必须标识为 Button ，Icon 为按钮图标样式，Desc 为提示文本；当设置文本框控件时，options 格式：｛Icon，Width，Desc，OnBeforeFocus，OnFocus，OnBlur，OnChange｝。 Array null
Items 项对象属性详细说明：
属性名 具体描述 类型
Text 面包屑项的显示文本。 String
Url 加载远程页面的URL。 String
Params 指定给加载远程页面的URL的参数对象。 String

html代码：

<div id="breadcrumbs" class="frame-breadcrumbs"></div>

javascript代码：

$("#breadcrumbs").BreadCrumbs({
    Items: [{ Text: "Home" }, { Text: "面包屑项1" }, { Text: "面包屑项2"}],
    ToolBar: [
        { Type: "TextBox", Icon: "icon-search", Desc: "搜索", Width: 150 },
        { Type: "Button", Icon: "icon-refresh", Desc: "刷新" }
    ],
    OnItemClick: function (sender, item) {
        $.Messager.Alert("点击项", item.Text, "info");
    }
});
