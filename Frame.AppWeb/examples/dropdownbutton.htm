DropDownButton

DropDownButton 用于创建下拉菜单按钮，通过对指定 div 元素使用 var ddButton = $("#目标元素id").DropDownButton(options); 这样一行代码，即可将目标元素生成为下拉菜单按钮。
有四种不同的尺寸：smaller，small，large。当不指定尺寸时，以标准尺寸显示。
若需要设置其他背景色，有7种不同的背景色：primary，info，error，success，warning，inverse，default。

DropDownButton 对象的 Options 属性详细说明见下表：
属性名 具体描述 类型 默认值
Text 按钮的显示文本。 String null
Size 按钮的尺寸。 String null
Mode 按钮菜单的背景色模式，有7种模式可选：info，primary，success，warning，default，error，inverse。 String primary
MenuItems 按钮的下拉菜单项对象集合。菜单项对象格式：{Text，Divider}。Divider 属性标识是否为分割线。 Array null
Disabled 设置一个值，该值标识菜单按钮是否禁用。 Boolean false
Options 事件属性对象详细说明见下表：
名称 参数 具体描述
OnItemClick text，idx 右键菜单菜单项的单击事件。事件传递菜单项的文本内容及其对应所在菜单中的索引值作为参数。
DropDownButton 对象方法说明见下表：
调用：$("id").Frame().方法名();
方法 参数 具体描述
Disable none 禁用下拉菜单按钮。
Enable none 启用下拉菜单按钮。

html代码：

<div id="btnGroup" class="frame-button-group"></div>
<div id="btnLargeGroup" class="frame-button-group"></div>
<div id="btnSmallGroup" class="frame-button-group"></div>
<div id="btnSmallerGroup" class="frame-button-group"></div>

javascript代码：

$("#btnGroup").DropDownButton({
    Text: "Action Button", Mode: "success",
    MenuItems: [
    { Text: "菜单项1" },
    { Text: "菜单项2" },
    { Text: "菜单项3" },
    { Divider: true },
    { Text: "菜单项4" }
    ],
    OnItemClick: function (text, idx) {
        $.Messager.Alert("系统消息", text, "info");
    }
});
