Pagination

Pagination 用于创建导航数据页面控件。它支持可配置选项页面导航，可以添加自定义按钮在正确的分页,增强其功能。
通过在 div 标签上添加 .frame-pagination 标识为分页控件。
通过使用 var page = $("#页面标签id").Pagination(options); 实现将目标元素转换为 Pagination 分页控件对象。

Options 属性对象详细说明见下表：
属性名 具体描述 类型 默认值
Total 总记录数，当初始化创建分页时必须设置。 Number 0
PageSize 每页显示的记录数。 Number 10
PageNumber 当分页创建完毕时初始显示的当前页码。 Number 1
Buttons 自定义功能按钮组。按钮对象格式：{Icon,Title,Color,OnClick,Separator}，其中 Icon 为按钮图标样式名称，Title 为按钮显示文本，Color 为按钮颜色样式，OnClick 为按钮事件，事件传递参数为 Title 。若只需要显示分隔，设置 Separator 为 true即可。 Array null
Options 事件属性对象详细说明见下表：
名称 参数 具体描述
OnSelectPage page，size 当用户进行翻页时触发，回调函数传递的2个参数分别为当前页码和每页显示记录数。
Pagination 对象方法说明见下表：
调用：$("id").Frame().方法名();
方法 参数 具体描述
Refresh total, pagenumber 根据总记录数和当前页码，刷新分页的页数。

html代码：

<div id="pageOfNoRecord"></div>
<div id="pageOfHasRecord"></div>
<div id="DpageOfHasTooliv3"></div>

javascript代码：

$("#pageOfNoRecord").Pagination({ Total: 0 });
$("#pageOfHasRecord").Pagination({ Total: 47,
    OnSelectPage: function (page, size) {
        $.Messager.Alert("系统消息", "page:" + page + ",size:" + size, "info");
    }
});
$("#pageOfHasTool").Pagination({ Total: 47,
    Buttons: [
        { Icon: "icon-trash", Title: "删除", Click: function (title) { alert(title); } },
        { Separator: true },
        { Icon: "icon-pencil", Title: "编辑", Click: function (title) { alert(title); } }
    ],
    OnSelectPage: function (page, size) {
        $.Messager.Alert("系统消息", "page:" + page + ",size:" + size, "info");
    }
});
