Combobox

Combobox 用于将正常的 select 元素转换为 Combobox 控件对象。通过在 select 标签添加 .frame-combobox 标识为 Frame 下拉框控件。
通过使用 var combobox = $("#页面标签id").Combobox(options); 这样一行代码，即可将目标元素通过 options 相关属性的配置生成相应的下拉框控件对象。

Options 属性对象详细说明见下表：
属性名 具体描述 类型 默认值
Editable 设置一个值，该值标识下拉框是否启用编辑功能。 Boolean true
Width 下拉框的宽度。 Number 100
PanelWidth 下拉框列表面板宽度。 Number null
PanelHeight 下拉框列表面板高度。 Number null
Mode 下拉框的背景色模式，有8种模式可选：info，primary，success，warning，default，error，inverse。 String primary
Value 绑定到下拉框的数据字段名称，该字段的值将作为下拉框的值。 String Key
Text 绑定到下拉框的数据字段名称，该字段的值将作为下拉框的显示文本。 String Value
DefaultText 显示的下拉框默认文本。 String 请选择
Datas 需要加载绑定到下拉框的数据集合。 Array null
Url 一个从远程服务器加载列表数据的URL。 String null
Method AJAX 请求方式 ("POST" 或 "GET")。 String POST
DataType 服务器返回的数据类型。可用值：xml，html，json，text。 String json
Params 发送到服务器的数据参数对象。将自动转换为请求字符串格式。GET 请求中将附加在 URL 后。 Object null
Disabled 设置一个值，该值标识下拉框是否禁用。 Boolean false
Options 事件属性对象详细说明见下表：
名称 参数 具体描述
OnBeforeSelect none 选中选项前触发。
OnSelected item，selection 选中选项后触发。接收参数：选中的元素，选中元素对应的数据对象。
OnSuccess datas 远程请求数据成功后触发。接收参数：服务器返回的数据对象。
OnError xhr，info，thrown 对远程服务器请求失败后触发。xhr：XMLHttpRequest对象；info：错误信息；thrown：捕获的异常对象。如果发生了错误，错误信息（info）除了得到 null 之外，还可能是 "timeout"，"error"，"notmodified"和"parsererror"。
OnComplete xhr，ts 远程请求服务器加载数据完成后回调函数。请求成功或失败之后均调用。
OnResolve datas 对远程请求获取到的数据或本地加载的数据进行格式处理的拓展事件。
OnBeforeOpen none 下拉框列表进行展开前触发。
Combobox 对象方法说明见下表：
调用：$("id").Frame().方法名();
方法 参数 具体描述
Disable none 禁用下拉框。
Enable none 启用下拉框。
Select which 选中指定索引或名称的项。
ToggleDropDown none 切换展开或闭合下拉框。
GetText none 获取选中选项文本值。
SetText text 设置选中选项文本值。
GetValue none 获取选中选项值。
SetValue value 设置选中选项值。
Clear none 重置下拉框。
Reload param 重新进行远程请求数据并绑定到下拉框。
LoadData datas 加载数据并绑定到下拉框。

html代码：

<select id="cmbDefault" class="frame-combobox">
    <option value="1">选项1</option>
    <option value="2">选项2</option>
    <option value="3">选项3</option>
</select>                 
<select id="cmbRemote" class="frame-combobox"></select>

javascript代码：

$("#cmbDefault").ComboBox({
    Disabled: false,
    PanelHeight: 100
});

$("#cmbRemote").ComboBox({
    Editable: false,
    Mode: "success",
    Method: "GET",
    Url: "../datas/combobox.json",
    Text: "Name",
    Value: "Value",
    PanelHeight: 100
});
