Navigaion

Navigation 是一个菜单导航控件，可以嵌入到页面的任何位置。通过在 ul 标签添加 .frame-navigation 标识为 Frame 菜单导航控件。
继而使用 var navigation = $("#页面标签id").Navigation(options); 这样一行代码，根据 options 相关属性的配置将目标元素生成为菜单导航控件对象。

Options 属性对象详细说明见下表：
属性名 具体描述 类型 默认值
Items 菜单项集合。设置该属性初始化菜单导航控件的内容。 Array null
Url 加载远程数据服务的URL，获取远程数据用来生成创建菜单控件的菜单项。 String null
Params 指定给加载远程页面的URL的参数对象。。 Object null
Method AJAX 请求方式 ("POST" 或 "GET")。 String POST
DataType 服务器返回的数据类型。可用值：xml，html，json，text。 String json
AutoExpand 设置一个值，该值标识菜单是否自动折叠闭合。 Boolean true
菜单项对象集合 Items 的菜单项对象属性详细说明：
属性名 具体描述 类型
Icon 菜单项图标样式名称。 String
Title 菜单项文本。 String
Children 菜单子菜单项集合。子菜单项对象格式：{ Title }。 Array
Options 事件属性对象详细说明见下表：
名称 参数 具体描述
OnHeaderClick sender，headItem 当菜单项没有子菜单项集合时，点击 Header 菜单项时触发。
OnChildirenClick sender，headItem，childItem 点击子菜单项时触发。
OnResolve datas 需要对远程加载获取到的数据作进一步的额外处理时，调用该事件函数。
OnBeforeExpand sender，headItem 展开 Header 菜单项时触发。
OnExpand sender，headItem 折叠或闭合 Header 菜单项时触发。
Navigation 对象方法说明见下表：
调用：$("id").Frame().方法名();
方法 参数 具体描述
Expand which 折叠或闭合指定菜单项。which 可以是菜单项的索引值或菜单项名称。
LoadData datas 加载本地菜单项集合数据。

html代码：

<ul id="nav" class="frame-navigation"></ul>
<ul id="navRemote" class="frame-navigation"></ul>

javascript代码：

$("#nav").Navigation({
    Items: [
    { Icon: "icon-dashboard", Title: "仪表盘" },
    { Icon: "icon-desktop", Title: "UI 元素",
        Children: [
            { Title: "元素标签" },
            { Title: "按钮 & 图标集" },
            { Title: "基本控件" }
        ]
    },
    { Icon: "icon-file-alt", Title: "其他页面" },
    { Icon: "icon-edit", Title: "表单" }
    ],
    OnHeaderClick: function (sender, item) {
        $.Messager.Alert("Header", item.Title, "info");
    },
    OnChildirenClick: function (sender, header, child) {
        $.Messager.Alert("Childiren", child.Title, "info");
    }
});

$("#navRemote").Navigation({
    Url: "../datas/navigation.json",
    Method: "GET",
    OnHeaderClick: function (sender, item) {
        $.Messager.Alert("Header", item.Title, "info");
    },
    OnChildirenClick: function (sender, header, child) {
        $.Messager.Alert("Childiren", child.Title, "info");
    }
});
