Resizable

Resizable 控件可用于将目标元素创建为一个可调整尺寸大小的元素，继而对目标元素的尺寸大小进行调整。
通过在 div 或其他块状标签上添加 .frame-resizable ， 标识为调整尺寸元素，通过使用 var resizable = $("#页面标签id").Resizable(options); 这样一行代码，即可将页面指定ID的标签通过 options 相关属性的配置生成相应的调整尺寸大小对象。

Options 属性对象详细说明见下表：
属性名 具体描述 类型 默认值
Handles 调整大小的鼠标缩放方向。 String n[north:北],e[east:东],s[south:南],w[west:西],ne[东北],se[东南],sw[西南],nw[西北]
Animate 是否使用动画效果。 Boolean false
Disabled 是否禁止调整目标元素的尺寸大小。如果设置为true将禁止调整元素尺寸。 Boolean false
MinWidth 目标元素可调整的尺寸最小宽度值。 Number 20
MinHeight 目标元素可调整的尺寸最小高度值。 Number 20
MaxWidth 目标元素可调整的尺寸最大宽度值。 Number 2000
MaxHeight 目标元素可调整的尺寸最大高度值。 Number 2000
Scope 元素可进行调整尺寸的边界值。 Number 3
Options 事件属性对象详细说明见下表：
名称 参数 具体描述
OnStartResize current[目标元素当前移动的信息对象]，e[mousedown鼠标事件对象] 当开始对目标元素进行调整尺寸大小时触发。
OnResize current[目标元素当前移动的信息对象]，e[mousemove鼠标事件对象] 当对目标元素的尺寸进行调整时触发。
OnStopResize current[目标元素当前移动的信息对象]，e[mouseup鼠标事件对象] 当停止对目标元素尺寸进行调整时触发。
OnEndResize e[mouseup鼠标事件对象] 当完成对目标元素尺寸调整时触发。
Resizable 对象方法说明见下表：
调用：$("id").Frame().方法名();
方法 参数 具体描述
Enable none 启用对目标元素进行尺寸调整的功能。
Disable none 禁用对目标元素进行尺寸调整的功能。

html代码：

<div id="resizable" style="width:100px;height:100px; top:10px; background-color:#87b87f;"></div>
<div id="resizeForLeft" style="width:100px;height:100px; top:110px; left:200px; background-color:#93cbf9; color:#fff;">只能向左拉动改变尺寸</div>
<div id="resizeForDown" style="width:100px;height:100px; top:210px; left:400px; background-color:#a069c3; color:#fff;">只能向下拉动改变尺寸</div>

javascript代码：

$("#resizable").Resizable();
$("#resizeForLeft").Resizable({ Handles: "w,sw,nw" });
$("#resizeForDown").Resizable({ Handles: "s,sw,se" });
只能向左拉动改变尺寸
只能向下拉动改变尺寸
