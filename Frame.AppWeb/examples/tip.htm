Tip

Tip 是一个信息提示控件，绑定在指定的目标元素上。若提示文本为固定信息，可通过在目标元素上指定 tip 属性，该属性的值将会被作为目标元素的提示信息文本。
继而使用 var tip = $("#页面标签id").Tip(options); 这样一行代码，根据 options 相关属性的配置为目标元素绑定提示信息控件对象。

Options 属性对象详细说明见下表：
属性名 具体描述 类型 默认值
Placement 定制提示信息内容显示的位置，默认为自动适应。位置可选：top，bottom，left，right。 String auto
Html 设置一个值，该值标识提示信息内容是否为HTML片段。 Boolean null
Title 目标元素的提示信息内容。内容的类型可为 HTML 片段、文本或使用回调函数返回的值。当目标元素已指定 tip 属性，则该属性失效。 String null
Trigger 触发显示提示信息文本的事件类型。有3种类型可选：hover,focus,click。 String hover
Mode 提示信息的背景色模式，有5种模式可选：info，success，warning，error，inverse。 String inverse
Tip 对象方法说明见下表：
调用：$("id").Frame().方法名();
方法 参数 具体描述
Toggle none 切换显示或关闭提示信息。
GetPosition none 获取目标元素的位置信息。
Show none 显示提示信息。
Close none 取消显示提示信息。

html代码：

<div id="tipDefault" style="border:1px solid #999; padding:2px;"  tip="冒泡提示文本">默认冒泡提示</div>
<div id="tipTop" style="border:1px solid #999; padding:2px;"  tip="冒泡提示文本">显示在顶部的冒泡提示</div>
<div id="tipBottom" style="border:1px solid #999; padding:2px;"  tip="冒泡提示文本">显示在底部冒泡提示</div>
<div id="tipLeft" style="border:1px solid #999; padding:2px;"  tip="冒泡提示文本">显示在左侧的冒泡提示</div>
<div id="tipRight" style="border:1px solid #999; padding:2px;"  tip="冒泡提示文本">显示在右侧的冒泡提示</div>

<div id="tipHtml" style="border:1px solid #999; padding:2px;">提示内容为HTML片段</div>
<div id="tipOnFunc"  style="border:1px solid #999; padding:2px;">使用Function返回内容作为提示文本</div>
<div id="tipOnClick" style="border:1px solid #999; padding:2px;">点击该目标元素时弹出提示文本</div>
<input id="txtOnFocus" type="text" class="frame-textbox" tip="焦点在文本框时的提示文本" />

javascript代码：

$("#tipDefault").Tip();
$("#tipTop").Tip({ Placement: "auto", Mode: "success" });
$("#tipBottom").Tip({ Placement: "bottom", Mode: "error" });
$("#tipLeft").Tip({ Placement: "left", Mode: "warning" });
$("#tipRight").Tip({ Placement: "right", Mode: "info" });

$("#tipHtml").Tip({ Placement: "auto", Html: true, Mode: "info", Title: "<div>HTML片段的提示内容</div>" });
$("#tipOnFunc").Tip({ Placement: "auto", Html: true, Mode: "warning",
    Title: function () { return "<div>这是通过 Funcion 返回的一段HTML内容</div>"; }
});
$("#tipOnClick").Tip({ Placement: "auto", Trigger: "click", Title: "单击时弹出的提示", Mode: "success" });
$("#txtOnFocus").Tip({ Placement: "auto right", Trigger: "focus", Mode: "warning" });
