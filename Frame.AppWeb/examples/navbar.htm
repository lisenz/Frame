NarBar

NavBar 是一个菜单工具栏控件，可以嵌入到页面的任何位置。通过在 div 标签添加 .frame-navbar 标识为 Frame 菜单工具栏控件。
继而使用 var navbar = $("#页面标签id").NavBar(options); 这样一行代码，根据 options 相关属性的配置将目标元素生成为菜单工具栏控件对象。

Options 属性对象详细说明见下表：
属性名 具体描述 类型 默认值
Brand 菜单项集合。设置该属性初始化菜单导航控件的内容。 Array null
NavItems 加载远程数据服务的URL，获取远程数据用来生成创建菜单控件的菜单项。 String null
UserMenu 指定给加载远程页面的URL的参数对象。 Object null
Brand 项对象属性详细说明：
属性名 具体描述 类型
Icon Brand的图标样式名称。 String
Text Brand文本。 String
NavItems 项集合中项对象的对象属性详细说明：
属性名 具体描述 类型
Icon 菜单项图标样式名称。 String
Text 菜单项文本。 String
Badge 菜单项的拓展徽章背景色模式。模式类型参照 Badge 控件的 Mode 属性。 String
Color 菜单项的背景色样式。有3种样式可选：grey，green，purple String
DropMenu 菜单子菜单项集合。 Array
DropMenu 集合的项对象的对象属性详细说明：
属性名 具体描述 类型
Header DropMenu 菜单的标题栏。标题栏对象格式为：｛Icon，Title｝。其中 Icon 为标题栏图标样式，Title 为标题文本。 Object
Footer DropMenu 菜单的底部操作项对象。对象格式为：｛Icon，Title｝。其中 Icon 为图标样式，Title 为内容文本。 String
Children DropMenu 菜单的菜单项集合。项对象格式为：｛Content，OnItemStyle｝，其中 Content 为项内容，可以是文本内容或HTML片段文本；OnItemStyle 是对 Content 内容做拓展处理的一个回调事件函数，返回处理后的新格式内容。 Array
OnLastItemClick DropMenu 菜单的底部操作项的单击事件，当点击底部操作项时触发。 String
UserMenu 项对象属性详细说明：
属性名 具体描述 类型
Photo 设置用户菜单项的用户头像图片的路径。 String
Content 用户菜单项的项内容。可以是内容文本抑或HTML片段文本。 String
Items 用户菜单项的子菜单项集合。项对象格式为：｛Icon，Text，Divider｝，其中Icon 为菜单项图标样式，Text为菜单项文本内容，Divider标识该子项是否为分隔线。 Array
Options 事件属性对象详细说明见下表：
名称 参数 具体描述
OnBrandClick sender 点击 Brand 项时触发。
OnBeforeItemOpen sender，item，index 菜单导航项展开前触发。当使用 return false 阻止单击事件。接收参数:当前点击的元素对象,元素对象对应的数据对象,数据对象在对象集合中的索引位置。
OnNavItemClick sender，item，index 点击菜单导航项时触发。
OnNavItemChildClick sender，index 点击展开的菜单导航项的子菜单项时触发。接收参数:当前点击的元素对象,元素对象在菜单列表中的索引位置。
OnBefornUserMenuOpen none 用户信息菜单 UserMenu 展开前触发。使用 return false 可阻止其单击事件。
OnUserMenuItemClick text 点击用户信息菜单的菜单项时触发。
NavBar 对象方法说明见下表：
调用：$("id").Frame().方法名();
方法 参数 具体描述
AddNavItem navitem 追加一个导航栏列表项。
AddNavItemDropMenu index, dropmenu 添加导航栏指定索引的列表项的菜单列表。
AddNavItemDropMenuItem index, dropmenuitems 添加导航栏指定索引的列表项的菜单的子菜单项。
SetNavItemText index, text 设置导航栏指定索引的列表项的菜单项文本。
DisableNavItem index 禁用指定索引对应的列表项。
EnableNavItem index 启用指定索引对应的列表项。
DisableNavDropMenuItem parent, index 禁用指定索引对应的列表项下的菜单中指定索引对应的菜单项。
EnableNavDropMenuItem parent, index 启用用指定索引对应的列表项下的菜单中指定索引对应的菜单项。
DisableUserMenu none 禁用用户信息菜单项。
EnableUserMenu none 启用用户信息菜单项。
DisableUserMenuItem index 禁用用户信息菜单中指定索引对应的菜单项。
EnableUserMenuItem index 启用用户信息菜单中指定索引对应的菜单项。

html代码：

<div id="navbar" class="frame-navbar"></div>

javascript代码：

$("#navbar").NavBar({
    Brand: { Icon: "icon-lemon", Text: "FrameUI" },
    NavItems: [
    {
        Icon: "icon-bell",
        Text: 4,
        Badge: "important",
        Color: "grey",
        DropMenu: {
            Header: {
                Icon: "icon-star-empty",
                Title: "消息列表"
            },
            Children: [
                {
                    Content: "消息记录1"
                },
                {
                    Content: "消息记录2"
                },
                {
                    Content: "消息记录3"
                },
                {
                    Content: "消息记录4"
                }
            ],
            Footer: {
                Icon: "icon-arrow-right",
                Text: "查看更多记录"
            },
            OnLastItemClick: function (navitem, index) {
                $.Messager.Alert("Footer", navitem.Text, "info");
            }
        }
    },
	{
	    Icon: "icon-cogs",
	    Color: "purple",
	    DropMenu: {
	        Header: {
	            Icon: "icon-cog",
	            Title: "配置列表"
	        },
	        Children: [
                {
                    Content: "<i class='icon-time'></i>更新日志"
                },
                {
                    Content: "<i class='icon-download green'></i>下载"
                },
                {
                    Content: "<i class='icon-question-sign orange'></i>关于"
                }
            ]
	    }
	}
    ],
    UserMenu: { Photo: "user-portrait", Content: "欢迎您,<br />Lisen",
        Items: [
            { Icon: "icon-cog", Text: "系统设置" },
            { Icon: "icon-user", Text: "个人资料" },
            { Divider: true },
            { Icon: "icon-off", Text: "注销" }
        ]
    },
    OnBeforeItemOpen: function (sender, item, index) {
        $.Messager.Alert("BeforeItemOpen", item.Text, "info");
        return true;
    },
    OnNavItemClick: function (sender, item, index) {
        $.Messager.Alert("NavItemClick", item.Text, "info");
    },
    OnNavItemChildClick: function (sender, index) {
        $.Messager.Alert("NavItemChildClick", $(sender).text(), "info");
    },
    OnBefornUserMenuOpen: function () {
        $.Messager.Alert("BefornUserMenuOpen", "BefornUserMenuOpen", "info");
        //return true;
    },
    OnUserMenuItemClick: function (text) {
        $.Messager.Alert("UserMenuItemClick", text, "info");
    }
});
