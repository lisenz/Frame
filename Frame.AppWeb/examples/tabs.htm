Tabs

Tabs 标签页控件可以当作半静态组件使用。通过在 div 标签添加 .frame-tabs 标识为 Frame 标签页控件容器。
该容器中可以在 HTML 页面放入多个 div 标签并设置指定属性，即可作为标签页对应的内容，引入js之后可以生成 Tabs 对象并做动态的切换。
通过使用 var tabs = $("#页面标签id").Tabs(options); 这样一行代码，即可将目标元素通过 options 相关属性的配置生成相应的标签页控件。

Options 属性对象详细说明见下表：
属性名 具体描述 类型 默认值
Tabs 选项卡面板定义的选项卡项对象集合。 Array null
Width 标签页控件的宽度。 Number auto
Height 标签页控件的高度。 Number auto
Plain 设置一个值，该值标识是否为简洁模式,即无边框。 Boolean false
Tabs 选项卡项对象的属性详细说明见下表：
属性名 具体描述 类型
Title 选项卡项的标题文本。 String
Url 一个URL用于加载远程数据,然后显示在面板。 String
Content 选项卡项对应的主板内容。该属性在已指定了 URL 属性的情况下将不生效。 String
Closable 设置一个值，该值标识是否显示关闭按钮。 Boolean
Selected 设置一个值，该值标识当前选项卡项是否默认被选中。 Boolean
Disabled 设置一个值，该值标识当前选项卡项是否禁用。 Boolean
Icon 设置选项卡项的显示图标。该图标对象格式为：{color,cls}，其中 color 指定图标的颜色，有16种颜色样式可选：dark，red，light-red，blue，light-blue，green，light-green，orange，light-orange，orange2，purple，pink，pink2，brown，grey，light-grey；cls 为图标样式名称。 Object
Options 事件属性对象详细说明见下表：
名称 参数 具体描述
OnLoad sender，content 当一个选项卡完成ajax远程载入对象时触发。
OnSelect sender，index 当用户选择一个选项卡时触发。
OnBeforeClose sender，index 在一个选项卡被关闭之前触发，返回false将取消关闭。
OnClose none 在用户关闭一个选项卡面板后触发。
OnAdd sender，index 在一个选项卡面板被添加后触发。
Button 对象方法说明见下表：
调用：$("id").Frame().方法名();
方法 参数 具体描述
Add tab 添加一个新的选项卡项。
AddRange array 批量添加多个选项卡项。
SelectTab which 根据标题或索引选择一个选项卡面板，which参数表示指定标题或索引的面板选被选中。
Exists which 验证一个特定的选项卡面板是否存在，参数which指向选项卡面板标题或索引。
GetTabIndex title 根据标题名称获取对应的选项卡面板索引。
GetSelectedTab none 获取当前被选中的选项卡面板。
Close which 关闭一个选项卡面板，which参数表示指定标题或索引的面板将被关闭。

html代码：

<div id="tab1" class="frame-tabs" style="margin:5px;">
    <div title="Append" style="padding:20px;">
    Html页面中已创建
    </div>
</div>
<div id="tab2" class="frame-tabs" style="margin:5px;">
    <div title="Append" icon="cls:'icon-home',color:'green'" style="padding:20px;">
    Html页面中已创建
    </div>
</div>
<div id="tab3" class="frame-tabs" style="margin:5px;">
    <div title="标签1" icon="cls:'icon-home',color:'green'" style="padding:20px;">
    内容1
    </div>
    <div title="标签2" icon="cls:'icon-food',color:'red'" style="padding:20px;">
    内容2
    </div>
</div>

javascript代码：

$("#tab1").Tabs({ Tabs: [
    { Title: "Content", Closable: true, Selected: true, Content: '<span style="padding:20px; display:inline-block;">指定Content属性来动态生成的内容</span>' },
    { Title: "Example", Closable: true, Url: "example.htm" }
]
});
$("#tab2").Tabs({ Tabs: [
    { Title: "Content", Closable: true, Selected: true, Icon: { cls: "icon-food", color: "red" }, Content: '<span style="padding:20px; display:inline-block;">指定Content属性来动态生成的内容</span>' },
    { Title: "Example", Closable: true, Url: "example.htm", Icon: { cls: "icon-cloud", color: "blue"} }
]
});
$("#tab3").Tabs({ Width: 700, Height: 200, Tabs:
[
    { Title: "Example", Closable: true, Url: "example.htm", Icon: { cls: "icon-cloud", color: "blue"} }
] 
});
