Button

Button 按钮控件可以当作静态组件使用。通过在 type = "button"的 button 标签添加 .frame-button 标识为 Frame 按钮控件对象。
若需要设置其他背景色，有8种不同的背景色模式：.frame-button-transparent，.frame-button-info，.frame-button-primary，.frame-button-success，.frame-button-warning，.frame-button-default，.frame-button-error，.frame-button-inverse 。
有四种不同的尺寸：.frame-button-smaller，.frame-button-small，.frame-button-normal，.frame-button-large，默认为标准尺寸。

还可以使用 Frame 组件形式生成 Button 控件，通过使用 var button = $("#页面标签id").Button(options); 这样一行代码，即可将目标元素通过 options 相关属性的配置生成相应的按钮控件。
Options 属性对象详细说明见下表：
属性名 具体描述 类型 默认值
Text 按钮的显示文本。 String null
Size 按钮的尺寸大小，有4种尺寸：smaller，small，normal，large。 String null
Mode 按钮的背景色模式，有8种模式可选：transparent，info，primary，success，warning，default，error，inverse。 String primary
CanBlock 设置一个值，该值标识按钮是否以 Block 形式占据页面整行显示。 Boolean false
Actived 设置一个值，该值标识按钮是否当前被激活状态。 Boolean false
Icon 按钮的图标 class 样式名称（若按钮需要图标的话）。 String/Array null
IconSite 按钮的图标在标签里的位置，有4种选项：left，both，right，only。注意：当属性为 both 时，Icon属性 为数组类型，且需要设置两个 class 样式。 String left
Width 按钮的宽度。 Number null
Disabled 设置一个值，该值标识按钮是否被禁用。 Boolean false
Options 事件属性对象详细说明见下表：
名称 参数 具体描述
OnClick sender 按钮的单击事件。
Button 对象方法说明见下表：
调用：$("id").Frame().方法名();
方法 参数 具体描述
Enable none 启用按钮。
Disable none 禁用按钮。
Actived active 设置按钮的激活状态，该状态表示当前按钮已被点击激活。

<button id="btnDefault" class="frame-button">Default Button</button>
<button id="btnPrimary" class="frame-button">Primary Button</button>
<button id="btnSuccess" class="frame-button">Success Button</button>
<button id="btnError" class="frame-button">Error Button</button>
<button id="btnWraning" class="frame-button">Warning Button</button>
<button id="btnInverse" class="frame-button">Inverse Button</button>
<button id="btnInfo" class="frame-button">Info Button</button>

<button id="btnBlock" class="frame-button">Block Button</button>

<button id="btnSmaller" class="frame-button">Smaller Button</button>

javascript代码：

$("#btnOriginal").Button({ Mode: "original" });
$("#btnDefault").Button({ Mode: "default" });
$("#btnPrimary").Button({ Mode: "primary" });
$("#btnSuccess").Button({ Mode: "success" });
$("#btnError").Button({ Mode: "error" });
$("#btnWraning").Button({ Mode: "warning" });
$("#btnInverse").Button({ Mode: "inverse" });
$("#btnInfo").Button({ Mode: "info" });

$("#btnBlock").Button({ Mode: "info", CanBlock: true });

$("#btnSmaller").Button({ Mode: "primary", Size: "smaller" });
$("#btnSmall").Button({ Mode: "success", Size: "small" });
$("#btnNormal").Button({ Mode: "error", Size: "normal" });
$("#btnLarge").Button({ Mode: "inverse", Size: "large" });

$("#btnIconOnLeft").Button({ Mode: "warning", Icon: "icon-bar-chart", IconSite: "left" });
$("#btnIconOnSide").Button({ Mode: "inverse", Icon: ["icon-plus", "icon-minus"], IconSite: "side" });
$("#btnIconOnRight").Button({ Mode: "info", Icon: "icon-bar-chart", IconSite: "right" });
$("#btnOnlyIcon").Button({ Mode: "success", Icon: "icon-bar-chart" });
