Badge

徽章控件可以当作静态组件使用。通过在 span 标签上添加 .frame-badge 。此时标签背景色默认显示灰底色。
若需要设置其他背景色，有7种不同的背景色：.frame-badge-info，.frame-badge-primary，.frame-badge-success，.frame-badge-warning，.frame-badge-important，.frame-badge-error，.frame-badge-inverse 。

还可以使用 Frame 组件形式生成 Badge 徽章，通过使用 var badge = $("#页面标签id").Badge(options); 这样一行代码，即可将页面指定ID的标签通过 options 相关属性的配置生成相应的徽章。
Options 属性对象详细说明见下表：
属性名 具体描述 类型 默认值
Text 徽章文本内容。当页面标签已有内容，若属性 Text 也设置文本，优先使用属性设置的文本内容。 String null
Mode 标签的背景色模式，有7种模式可选：info，primary，success，warning，important，error，inverse。 String primary

html代码：

<span id="bdPrimary" class="frame-badge">1</span>
<span id="bdInfo" class="frame-badge">2</span>
<span id="bdSuccess" class="frame-badge">3</span>
<span id="bdError" class="frame-badge">4</span>
<span id="bdInverse" class="frame-badge">5</span>
<span id="bdWarning" class="frame-badge">6</span>
<span id="bdImportant" class="frame-badge">7</span>

javascript代码：

$("#bdPrimary").Badge({ Mode: "primary" });
$("#bdInfo").Badge({ Mode: "info" });
$("#bdSuccess").Badge({ Mode: "success" });
$("#bdError").Badge({ Mode: "error" });
$("#bdInverse").Badge({ Mode: "inverse" });
$("#bdWarning").Badge({ Mode: "warning" });
$("#bdImportant").Badge({ Mode: "important" });
