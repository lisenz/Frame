ContextMenu

用于为目标元素创建上下文右键菜单，通过使用 var contextmenu = $("#目标元素id").ContextMenu(options); 这样一行代码，即可在目标元素的可见区域中使用右键菜单。
需要使用右键菜单的目标元素，样式必须使用 position:relative; 。

Options 属性对象详细说明见下表：
属性名 具体描述 类型 默认值
Target 右键菜单所依附的目标元素。 Selector null
Width 右键菜单的宽度。 Number null
Mode 菜单的背景色模式，有7种模式可选：info，primary，success，warning，default，error，inverse。 String primary
Items 右键菜单的菜单项对象集合。菜单项对象格式：{Text，HasChild，Childs:[]，Icon，IconColor，Divider，Disabled，Actived，OnClick}。 Array null
X 菜单在区域中的 X 坐标位置。 Number 0
Y 菜单在区域中的 Y 坐标位置。 Number 0
菜单项对象集合 Items 的菜单项对象属性详细说明：
属性名 具体描述 类型 默认值
Text 菜单项文本。 String null
HasChild 标识是否有子项。 Boolean null
Childs 菜单项的子项对象集合。当 HasChild 属性为 true ，该属性才有效。 Array null
Icon 菜单项显示图标样式名称。 String null
IconColor 当菜单项使用图标时，设置图标颜色的样式。有16种颜色样式可选：dark，red，light-red，blue，light-blue，green，light-green，orange，light-orange，orange2，purple，pink，pink2，brown，grey，light-grey。 Selector null
Divider 设置一个值，该值标识当前菜单项是否为分割线。 Boolean null
Disabled 设置一个值，该值标识菜单项是否禁用。 Boolean null
Actived 设置一个值，该值标识菜单项是否已被点击激活。 Boolean null
OnClick 右键菜单菜单项的单击事件。事件参数传递当前单击的菜单项元素对象和菜单项对象。 sender，item null
Options 事件属性对象详细说明见下表：
名称 参数 具体描述
OnItemClick sender，item 右键菜单菜单项的单击事件。该事件在菜单项未设置其自身的 OnClick 事件时生效。
ContextMenu 对象方法说明见下表：
调用：$("id").Frame().方法名();
方法 参数 具体描述
Show option 显示右键菜单。
Close none 关闭右键菜单。
EnableItem lv，idx 启用指定层级下指定索引的菜单项。其中参数 lv 标识菜单的层级数，idx 标识该层级下的菜单项索引。
DisableItem lv，idx 禁用指定层级下指定索引的菜单项。其中参数 lv 标识菜单的层级数，idx 标识该层级下的菜单项索引。
DeactiveItem lv，idx 取消指定层级下指定索引的菜单项激活状态，该状态表示该菜单项已被取消激活。
ActiveItem lv，idx 设置指定层级下指定索引的菜单项激活状态，该状态表示该菜单项已被点击激活。

html代码：

<div id="cm" style="width:600px; height:300px; border:1px solid #333; color:#999; position:relative;">
在该区域中使用右键菜单
</div>

javascript代码：

$("#cm").ContextMenu({
    Items: [
        { Text: "菜单项-带图标", Icon: "icon-leaf", IconColor: "blue",
            OnClick: function (sender, o) {
                $.Messager.Alert("菜单项文本", o.Text, "info");
            }
        },
        { Text: "菜单项-已点击且禁用", Disabled: true, Actived: true },
        { Text: "菜单项-有子项", HasChild: true, Icon: "icon-leaf", IconColor: "blue",
            Childs: [
                { Text: "二级菜单项" },
                { Text: "二级菜单项-已点击", Actived: true },
                { Text: "二级菜单项-带图标", Icon: "icon-leaf", IconColor: "blue" },
                { Divider: true },
                { Text: "二级菜单项" }
            ]
        },
        { Divider: true },
        { Text: "菜单项" }
    ],
    OnItemClick: function (sender, o) {
        $.Messager.Alert("菜单项文本", o.Text, "info");
    }
});
在该区域中使用右键菜单
