DatePicker

DatePicker 是一个日期控件。通过绑定关联 type="text" 的 input 控件使其允许用户选择一个日期。输入一个字符串到文本框中能够转换为一个有效日期值，选择日期也可以被格式化为指定格式的格式。
继而使用 var datepicker = $("#页面标签id").DatePicker(options); 这样一行代码，根据 options 相关属性的配置将目标元素生成为日期控件对象。

Options 属性对象详细说明见下表：
属性名 具体描述 类型 默认值
Width 日期控件的宽度。 Number 165
ShowType 日期控件的显示控件类型。有两种类型可选：ymd，ym。 String ymd
DateFormat 日期格式。 String yyyy-MM-dd hh:mm
IsShowDate 设置一个值，该值指示是否默认显示日期。 Boolean true
CanShowTime 设置一个值，该值指示是否显示时间。 Boolean true
Disabled 设置一个值，该值标识是否禁用日期控件。 Boolean false
ReadOnly 设置一个值，该值标识日期控件是否为只读模式。 Boolean false
Options 事件属性对象详细说明见下表：
名称 参数 具体描述
OnExtractor date 对选中的日期值作进一步格式处理，返回处理后的日期值。
OnChange date 当选择的日期值发生改变时触发。
OnResolve date 需要对发生改变的日期值再做进一步格式处理时，可以设置该事件。该事件贯穿所有触发条件。
DatePicker 对象方法说明见下表：
调用：$("id").Frame().方法名();
方法 参数 具体描述
SetValue value 设置日期值。
GetValue none 获取日期值。
Disable content 禁用日期控件。
Enable none 启用日期控件。

html代码：

<input id="dpDefault" class="frame-datepicker" type="text" />
<input id="dpNoTime" class="frame-datepicker" type="text" />
<input id="dpYmd" class="frame-datepicker" type="text" />
<input id="dpYm" class="frame-datepicker" type="text" />

javascript代码：

$("#dpDefault").DatePicker();
$("#dpNoTime").DatePicker({ CanShowTime: false });
$("#dpYmd").DatePicker({ DateFormat: "yyMMdd", CanShowTime: false,
    OnResolve: function (val) {
        var year = (new Date()).getFullYear().toString().substring(0, 2) + val.substring(0, 2);
        var month = val.substring(2, 4);
        var day = val.substring(4, 6);
        return year + "-" + month + "-" + day;
    }
});
$("#dpYm").DatePicker({ ShowType: "ym" });
