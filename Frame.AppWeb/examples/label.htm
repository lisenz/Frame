Label

标签控件可以当作静态组件使用。通过在 span 或者 label 标签上添加 .frame-label 。此时标签背景色默认显示灰底色。
若需要设置其他背景色，有8种不同的背景色：.frame-label-transparent，.frame-label-info，.frame-label-primary，.frame-label-success，.frame-label-warning，.frame-label-important，.frame-label-error，.frame-label-inverse 。
有三种不同的尺寸：.frame-label-small，.frame-label-large，.frame-label-xlarge，默认为标准尺寸。

如果要使标签带箭头，需要对 .arrowed，.arrowed-in，.arrowed-right，.arrowed-in-right 进行组合使用，具体请参考下表：
组合 class 类 组合效果
.arrowed：<span class="frame-label frame-label-primary arrowed">Arrowed Label</span> Arrowed Label
.arrowed-in：<span class="frame-label frame-label-primary arrowed-in">Arrowed-in Label</span> Arrowed-in Label
.arrowed-right：<span class="frame-label frame-label-primary arrowed-right">Arrowed-right Label</span> Arrowed-right Label
.arrowed-in-right：<span class="frame-label frame-label-primary arrowed-in-right">Arrowed-in-right Label</span> Arrowed-in-right Label
.arrowed，.arrowed-in-right：<span class="frame-label frame-label-primary arrowed arrowed-in-right">Arrowed left Label</span> Arrowed left Label
.arrowed-in，.arrowed-right：<span class="frame-label frame-label-primary arrowed-in arrowed-right">Arrowed right Label</span> Arrowed right Label
.arrowed，.arrowed-right：<span class="frame-label frame-label-primary arrowed-in arrowed-right">Arrowed both Label</span> Arrowed both Label
.arrowed-in，.arrowed-in-right：<span class="frame-label frame-label-primary arrowed-in arrowed-right">Arrowed in both Label</span> Arrowed in both Label
还可以使用 Frame 组件形式生成 Label 标签，通过使用 var lbl = $("#页面标签id").Label(options); 这样一行代码，即可将页面指定ID的标签通过 options 相关属性的配置生成相应的标签。
Options 属性对象详细说明见下表：
属性名 具体描述 类型 默认值
Text 标签文本内容。当页面标签已有内容，若属性 Text 也设置文本，优先使用属性设置的文本内容。 String null
Size 标签尺寸，有3种尺寸：small，large，xlarge。normal 表示默认标准尺寸。 String normal
Mode 标签的背景色模式，有8种模式可选：transparent，info，primary，success，warning，important，error，inverse。 String primary
WithArrow 标识标签箭头的样式，有8种样式可选：arrowed-left，arrowed-both，arrowed-right，arrowed-in-left，arrowed-in-both，arrowed-in-right，arrow-left，arrow-right 。若标签不需要有箭头样式，设置为 false。 String false
Icon 标签的图标 class 样式名称（若标签需要图标的话）。 String/Array null
IconSite 标签的图标在标签里的位置，有4种选项：left，both，right，only。注意：当属性为 both 时，Icon属性 为数组类型，且需要设置两个 class 样式。 String left
Width 标签的宽度。若显式设置标签宽度，则标签根据内容自适应宽度。 Number null

javascript代码：

$("#lbDefault").Label();
$("#lbTransparent").Label({ Mode: "transparent" });
$("#lbArrowedLeft").Label({ WithArrow: "arrowed-left" });
$("#lbArrowedBoth").Label({ WithArrow: "arrowed-both" });
$("#lbArrowedRight").Label({ WithArrow: "arrowed-right" });
$("#lbArrowedInLeft").Label({ WithArrow: "arrowed-in-left" });
$("#lbArrowedInBoth").Label({ WithArrow: "arrowed-in-both" });
$("#lbArrowedInRight").Label({ WithArrow: "arrowed-in-right" });
$("#lbArrowLeft").Label({ WithArrow: "arrow-left" });
$("#lbArrowRight").Label({ WithArrow: "arrow-right" });
//---------------------------------------------------------------
$("#lbPrimary").Label({ Mode: "info" });
$("#lbInfo").Label({ Mode: "primary" });
$("#lbSuccess").Label({ Mode: "success" });
$("#lbWarning").Label({ Mode: "warning" });
$("#lbImportant").Label({ Mode: "important" });
$("#lbError").Label({ Mode: "error" });
$("#lbInverse").Label({ Mode: "inverse" });
//---------------------------------------------------------------
$("#lbNormal").Label({ Size: "normal" });
$("#lbSmall").Label({ Size: "small" });
$("#lbLarge").Label({ Size: "large" });
$("#lbXLarge").Label({ Size: "xlarge" });
//---------------------------------------------------------------

$("#lbOnlyIcon").Label({ Icon: "icon-cogs ", IconSite: "only", Size: "xlarge" });
