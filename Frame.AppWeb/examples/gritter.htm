Gritter

Gritter 是一个消息通知控件，提供不同类型的消息框。提供了几个方法以便轻松为界面创建消息。

Gritter 对象方法说明见下表：
调用：$.Gritter.方法名();
方法 参数 具体描述
Add item 在界面创建一个指定配置的消息。item 为消息对象，对象格式为：｛Title，Text，Image，Sticky，Time，Cls｝。其中，Title 为消息界面标题文本；Text 为消息内容；Image 为消息图片路径；Sticky 为标识消息是否在设定时间内消失；Time 为消息消失的预设时间，与Sticky属性相互使用；Cls 为消息背景色模式，有5种模式可选：[info，error，success，warning，light]。
Remove index 移除指定索引值对应的消息。
Clear none 移除所有消息。

html代码：

<button id="btn1" class="frame-button frame-button-inverse">Gritter On Image</button>
<button id="btn2" class="frame-button frame-button-inverse">Gritter No Image</button>
<button id="btn3" class="frame-button frame-button-inverse">Gritter Title</button>

javascript代码：

$("#btn1").click(function () {
    $.Gritter.Add({
        Text: "Lorem ipsum dolor sit amet, consectetur adipiscing elit. Vivamus eget tincidunt velit. Cum sociis natoque penatibus et magnis dis parturient montes, nascetur ridiculus mus.",
        Sticky: true,
        Time: 5000,
        Cls: "error",
        Title: "标题",
        Image: "../images/user.jpg"
    });
});

$("#btn2").click(function () {
    $.Gritter.Add({
        Text: "Lorem ipsum dolor sit amet, consectetur adipiscing elit. Vivamus eget tincidunt velit. Cum sociis natoque penatibus et magnis dis parturient montes, nascetur ridiculus mus.",
        Sticky: false,
        Time: 5000
    });
});

$("#btn3").click(function () {
