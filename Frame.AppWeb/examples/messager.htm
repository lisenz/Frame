Messager

Messager 是一个消息控件，提供不同类型的消息框。用户可以使用 Messager 回调函数做交互操作。

Messager 对象方法说明见下表：
调用：$.Messager.方法名();
方法 参数 具体描述
Alert title, message, icon, fn 显示一个 Alert 窗体消息框。其中 title 为标题文本；message 为消息内容；icon 为显示的图标，可选值为：info，error，warn，ask，exclamation；fn 为窗体关闭时触发的回调函数。
Confirm title, message, fn 显示一个 Confirm 窗体提示框。其中 title 为标题文本；message 为消息内容；fn 为窗体关闭时触发的回调函数。
Show options 显示一个消息窗体。options属性对象格式：｛Title，Message，Width，Height｝。

html代码：

<button id="btn1" class="frame-button frame-button-success">Alert Error</button>
<button id="btn2" class="frame-button frame-button-success">Alert Info</button>
<button id="btn3" class="frame-button frame-button-success">Alert Warn</button>
<button id="btn4" class="frame-button frame-button-success">Alert Question</button>
<button id="btn5" class="frame-button frame-button-success">Alert Exclamation</button>
<button id="btn6" class="frame-button frame-button-success">Confirm</button>
<button id="btn7" class="frame-button frame-button-success">Show</button>

javascript代码：

$("#btn1").on("click", function () {
    $.Messager.Alert("Alert", "提示消息", "error");
});

$("#btn2").on("click", function () {
    $.Messager.Alert("Alert", "提示消息", "info");

});
$("#btn3").on("click", function () {
    $.Messager.Alert("Alert", "提示消息", "warn");

});
$("#btn4").on("click", function () {
    $.Messager.Alert("Alert", "提示消息", "ask");

});
$("#btn5").on("click", function () {
    $.Messager.Alert("Alert", "提示消息", "exclamation");

});
$("#btn6").on("click", function () {
    $.Messager.Confirm("Alert", "是否继续？", function (r) {
        if (r) {
            $.Messager.Alert("Alert", "是", "info");
        } else {
            $.Messager.Alert("Alert", "否", "info");
        }
    });

});
$("#btn7").on("click", function () {
    $.Messager.Show({ Title: "消息", Message: "消息内容" });

});
