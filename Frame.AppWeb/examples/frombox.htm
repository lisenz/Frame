FormBox

FormBox 是一个窗体控件，可以利用嵌入到页面的任何位置的 div 标签，继而使用 var formbox = $("#页面标签id").FormBox(options); 这样一行代码或使用 js 代码，根据 options 相关属性的配置将目标元素生成为窗体控件对象。

Options 属性对象详细说明见下表：
属性名 具体描述 类型 默认值
HasHeader 设置一个值，该值标识窗体是否使用标题栏。 Boolean true
Closable 设置一个值，该值标识窗体是否显示关闭按钮。 String null
Commands 设置窗体标题栏的拓展功能按钮集合。 Array null
Title 窗体标题栏的标题文本。 String 标题
Icon 窗体标题栏的图标样式名称。 String icon-desktop
HasFooter 设置一个值，该值标识窗体是否使用底部页脚功能。 Boolean true
Buttons 设置窗体底部页脚的功能按钮集合。 Array null
Left 设置窗体的left位置。 Number auto
Top 设置窗体的top位置。 Number auto
CloseOnDestroy 设置一个值，该值标识窗体关闭时是否同时从组件管理器中移除。 Boolean false
CanDraggable 设置一个值，该值标识窗体是否支持拖动。 Boolean false
CanResizable 设置一个值，该值标识窗体尺寸是否支持改变。 Boolean false
IsModal 设置窗体显示时是否使用遮罩层。 Boolean false
Url 设置加载远程页面的URL。 String null
Params 指定给加载远程页面的URL的参数对象。 Object null
Content 设置窗体的主体内容。主体内容格式可以为字符串或HTML片段 String null
Width 窗体的宽度。 Number auto
Height 窗体的高度。 Number auto
Options 事件属性对象详细说明见下表：
名称 参数 具体描述
OnBeforeShow none 显示窗体前触发。
OnShown none 显示窗体时触发。
OnBeforeClose none 关闭窗体前触发。
OnClose none 关闭窗体时触发。
FormBox 对象方法说明见下表：
调用：$("id").Frame().方法名();
方法 参数 具体描述
Show position 显示窗体。默认显示窗体时，位置居中浏览器。可以通过指定 position 使窗体的显示位置发生改变。
Close none 关闭窗体。注意，这里关闭窗体，但实际窗体仍然存在于组件管理器中。
Destroy none 彻底关闭窗体，此时窗体从组件管理器中销毁。
Load url, params 设置远程页面的URL和传递的参数，加载到窗体的主体位置。
Reload params 传递新的参数，重新加载远程页面。

html代码：

<button id="btnFromBox1" class="frame-button frame-button-primary">FormBox Has Content</button>
<button id="btnFromBox2" class="frame-button frame-button-primary">FormBox On Remote</button>
<div id="formbox1">
    <form>
        <div class="frame-form-group">
            <label class="col-4 control-label">账户名称：</label>
            <div class="col-4">
                <input type="text" />
            </div>
        </div>
        <div class="space-4"></div>
        <div class="frame-form-group">
            <label class="col-4 control-label">登录密码：</label>
            <div class="col-4">
                <input type="password" />
            </div>
        </div>
    </form>
</div>

javascript代码：

$("#formbox1").FormBox({
    Title: "Form",
    CanDraggable: true,
    Closable: false,
    CanResizable: true,
    IsModal: true,
    Width: 350,
    Height: 250,
    Buttons: [
    { Mode: "info", Icon: "icon-ok", Text: "确定" },
    { Mode: "error", Icon: "icon-power-off", Orientation: "right", Text: "取消", OnClick: function () { $.Frame.Get("formbox1").Close(); } }]
});

$("#btnFromBox1").click(function () {
    $.Frame.Get("formbox1").Show();
    return false;
});

$("#btnFromBox2").click(function () {
    $.FormBox.Show({
        Title: "Remote",
        HasFooter: false,
        CanDraggable: true,
        Closable: true,
        CanResizable: true,
        Url: "example.htm",
        IsModal: true,
        Width: 400,
        Height: 300
    });
    return false;
});
