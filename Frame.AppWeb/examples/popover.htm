Popover

Popover 是一个消息提示弹窗控件，绑定在指定的目标元素上。弹窗控件默认为隐藏状态。
通过使用 var popover = $("#页面标签id").Popover(options); 这样一行代码，根据 options 相关属性的配置为目标元素绑定消息提醒弹窗控件对象。

Options 属性对象详细说明见下表：
属性名 具体描述 类型 默认值
Placement 定制消息提示弹窗显示的位置，默认为自动适应。位置可选：top，bottom，left，right。 String auto
Content 消息提示弹窗的消息内容。内容的类型可为 HTML 片段、文本或使用回调函数返回的值。 String null
Title 消息提示弹窗的标题内容。内容的类型可为 HTML 片段、文本或使用回调函数返回的值。 String null
Width 消息提示弹窗的宽度。 Number null
Mode 消息提示弹窗的背景色模式，有5种模式可选：info，success，warning，error。 String null
Popover 对象方法说明见下表：
调用：$("id").Frame().方法名();
方法 参数 具体描述
Toggle none 切换显示或关闭消息提示弹窗控件。
IsVisible none 返回一个值，该值标识消息提示弹窗控件是否可见。
Reset content 重置消息提示弹窗的消息内容。
Show none 显示消息提示弹窗控件。
Close none 取消显示消息提示弹窗控件。

html代码：

<button id="popDefault" class="frame-button frame-button-primary">Default Popover</button>                    
<button id="popError" class="frame-button frame-button-primary">Error Popover</button>
<button id="popWarning" class="frame-button frame-button-primary">Warning Popover</button>
<button id="popInfo" class="frame-button frame-button-primary">Info Popover</button>
<button id="popSuccess" class="frame-button frame-button-primary">Success Popover</button>
<button id="popNoTitle" class="frame-button frame-button-primary">NoTitle Popover</button>
<button id="popHasIcon" class="frame-button frame-button-primary">Title Has Icon Popover</button>

javascript代码：

$("#popDefault").Popover({ Title: "标题", Content: "默认弹出框提醒控件<br/>指定标题" });
$("#popSuccess").Popover({ Mode: "success", Placement: "left", Title: "标题", Content: "向左侧弹出框提醒控件<br/>success 背景模式" });
$("#popError").Popover({ Mode: "error", Placement: "right", Title: "标题", Content: "向右侧弹出框提醒控件<br/>error 背景模式" });
$("#popWarning").Popover({ Mode: "warning", Placement: "top", Title: "标题", Content: "在顶部弹出框提醒控件<br/>warning 背景模式" });
$("#popInfo").Popover({ Mode: "info", Placement: "bottom", Title: "标题", Content: "在底部弹出框提醒控件<br/>info 背景模式" });
$("#popNoTitle").Popover({ Mode: "notitle", Content: "默认弹出框提醒控件<br/>无标题栏的提醒框" });
$("#popHasIcon").Popover({ Mode: "error", Title: '<i class="icon-bolt red"></i> 标题', Content: "默认弹出框提醒控件<br/>标题栏带图标的提醒框" });
