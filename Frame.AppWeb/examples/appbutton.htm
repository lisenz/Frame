AppButton

AppButton 用于创建 App 按钮，通过对指定 button 元素使用 var app = $("#目标元素id").AppButton(options); 这样一行代码，即可将目标元素生成为 App 按钮。
有三种不同的尺寸：normal，small。当不指定尺寸时，以标准尺寸显示。
若需要设置其他背景色，有7种不同的背景色：primary，info，error，success，warning，inverse，default。

AppButton 对象的 Options 属性详细说明见下表：
属性名 具体描述 类型 默认值
Text 按钮的显示文本。 String null
Size 按钮的尺寸。 String null
Mode 按钮菜单的背景色模式，有7种模式可选：info，primary，success，warning，default，error，inverse。 String primary
Icon 按钮的图标样式。菜单项对象格式：{Text，Divider}。Divider 属性标识是否为分割线。 Array null
BookMark 设置按钮的书签对象。对象类型允许为 Label 或 Badge。 object null
BookMarkSite 按钮的书签位置。 String null
Disabled 设置一个值，该值标识 App 按钮是否禁用。 Boolean false
Options 事件属性对象详细说明见下表：
名称 参数 具体描述
OnClick sender 按钮的单击事件。
AppButton 对象方法说明见下表：
调用：$("id").Frame().方法名();
方法 参数 具体描述
Disable none 禁用按钮。
Enable none 启用按钮。
SetBookMarkText text 设置按钮书签的文本内容。

html代码：

<button id="app" class="frame-button-app">AppButton</button>
<button id="appNormal" class="frame-button-app">AppButton</button>
<button id="appSmall" class="frame-button-app">Button</button>
<button id="appOnlyIcon" class="frame-button-app"></button>    
<button id="appOnlyText" class="frame-button-app">Button</button>
<button id="appWithLabel" class="frame-button-app">Label</button>       
<button id="appWithBadge" class="frame-button-app">Badge</button>

javascript代码：

$("#app").AppButton({ Icon: "icon-home", OnClick: function (sender) { $.Messager.Alert("系统消息", $(sender).text(), "info"); } });
$("#appNormal").AppButton({ Icon: "icon-home", Size:"normal", OnClick: function (sender) { $.Messager.Alert("系统消息", $(sender).text(), "info"); } });
$("#appSmall").AppButton({ Icon: "icon-home", Size: "small", OnClick: function (sender) { $.Messager.Alert("系统消息", $(sender).text(), "info"); } });

$("#appOnlyIcon").AppButton({ Icon: "icon-home", Mode: "success" });
$("#appOnlyText").AppButton({ Mode: "error" });

$("#appWithLabel").AppButton({ Icon: "icon-home", Mode: "warning", BookMark: $("<label></label>").Label({ WithArrow: "arrowed-in-left", Mode: "success", Text: "推荐" }) });
$("#appWithBadge").AppButton({ Icon: "icon-home", Mode: "warning", BookMark: $("<span></span>").Badge({ Mode: "success", Text: "6" }) });
