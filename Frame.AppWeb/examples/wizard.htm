Wizard

Wizard 用于设计创建一个向导界面。通过对 div 添加 .frame-wizard 样式，标识为 Frame 的向导控件。
通过 Frame 组件形式生成向导控件对象，使用 var wizard = $("#页面标签id").Wizard(options); 实现将目标元素转换为 Wizard 控件对象。

Options 属性对象详细说明见下表：
属性名 具体描述 类型 默认值
Steps 指定每个步骤的配置对象信息。配置属性为：｛Step，StepTarget，Title｝。其中 Step 标识步骤的索引；StepTarget 指定该步骤的内容主体容器的ID值；Title 为步骤的标题文本。 Array null
WizardTarget 指定包含每个步骤主体内容容器的主容器的ID。 String null
ActionTarget 指定向导操作区域的容器ID。 String null
Options 事件属性对象详细说明见下表：
名称 参数 具体描述
OnActive sender，step 当前步骤被激活时触发。sender 参数标识当前步骤的容器元素对象，step 为步骤配置对象。
OnComplete sender，step 步骤完成时触发。sender 参数标识当前步骤的容器元素对象，step 为步骤配置对象。
Radio 对象方法说明见下表：
调用：$("id").Frame().方法名();
方法 参数 具体描述
Previous none 上一步骤。
Next none 下一步骤。

javascript代码：

$("#frame-wizard").Wizard({
    Steps: [
    { Step: 1, StepTarget: "#step1", Title: "第一步" },
    { Step: 2, StepTarget: "#step2", Title: "第二步" },
    { Step: 3, StepTarget: "#step3", Title: "第三步" },
    { Step: 4, StepTarget: "#step4", Title: "第四步" }
    ],
    WizardTarget: "#step-container",
    ActionTarget: "#action-container",
    OnActive: function (sender, e) {
        $.Messager.Alert("Step", "Active Step" + e.Step, "info");
    },
    OnComplete: function (sender, e) {
        $.Messager.Alert("Step", "Complete Step" + e.Step, "info");
    }
});

$("#btnPrev").click(function () {
    $("#frame-wizard").Frame().Previous();
});
$("#btnNext").click(function () {
    $("#frame-wizard").Frame().Next();
});
