Spinner

Spinner 用于创建调节器控件。它是将正常的 type = "text" 的 input 控件转换为 Spinner 控件对象。
通过 Frame 组件形式生成 Spinner 控件，使用 var spinner = $("#页面标签id").Spinner(options); 实现将目标元素转换为 Spinner 控件对象。

Options 属性对象详细说明见下表：
属性名 具体描述 类型 默认值
Type 调节器的值类型。有三种类型：float，int，time。 String float
Value 调节器的文本值。 Number/String null
Width 调节器的宽度。 Number 100
ReadOnly 设置一个值，该值标识调节器是否为只读。 Boolean false
Disabled 设置一个值，该值标识调节器是否为禁用状态。 Boolean false
Step 每次递增或递减的步长值。注意：这里步长值必须与 Type 属性所定义的数据类型一致。 Number 0.1
IsNegative 设置一个值，该值标识调节器是否只能为负数。 Boolean false
Place 小数保留位数,当Type=float时生效。 Number 2
MinValue 调节器允许的最小值。 Number 0
MaxValue 调节器允许的最大值。 Number 100
Options 事件属性对象详细说明见下表：
名称 参数 具体描述
OnChange sender 当调节器的值发生改变时触发。
Spinner 对象方法说明见下表：
调用：$("id").Frame().方法名();
方法 参数 具体描述
Enable none 启用调节器。
Disable none 禁用调节器。
SetValue value 设置调节器的值。
GetValue none 获取调节器的当前值。

html代码：

// 默认调节器
<input id="txtSpinner" class="frame-spinner" type="text" />
// 整数调节器
<input id="txtSpinnerForInt" class="frame-spinner" type="text" />
// 时间调节器
<input id="txtSpinnerForTime" class="frame-spinner" type="text" />
// 禁用状态
<input id="txtDisabled" class="frame-spinner" type="text" />

javascript代码：

$("#txtSpinner").Spinner({ OnChange: function (value) { $.Messager.Alert("系统消息", value, "info"); } });
$("#txtSpinnerForInt").Spinner({ Type: "int", Step: 1, OnChange: function (value) { $.Messager.Alert("系统消息", value, "info"); } });
$("#txtSpinnerForTime").Spinner({ Type: "time", Step: 1, OnChange: function (value) { $.Messager.Alert("系统消息", value, "info"); } });
$("#txtDisabled").Spinner({ Disabled: true });
