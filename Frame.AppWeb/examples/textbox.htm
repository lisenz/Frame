TextBox

TextBox 文本框控件可以当作静态组件使用。通过在 type = "text"的 input 标签或者 textarea 标签上添加 .frame-textbox 标识为 Frame 文本框控件对象。
有六种不同的尺寸：.input-mini，.input-small，.input-medium，.input-large，.input-xlarge，.input-xxlarge，默认为标准尺寸。

还可以使用 Frame 组件形式生成 TextBox 控件，通过使用 var textbox = $("#页面标签id").TextBox(options); 这样一行代码，即可将目标元素通过 options 相关属性的配置生成相应的文本框控件。
Options 属性对象详细说明见下表：
属性名 具体描述 类型 默认值
Icon 设置文本框的内联图标的样式名称。 String null
Size 文本框的尺寸，有6种尺寸：mini，small，medium，large，xlarge，xxlarge。medium 表示默认标准尺寸。 String medium
IconSite 文本框中内联图标的位置，有两个位置方向：left，right。此属性若要生效，必须设置 Icon 属性。 String left
IconColor 文本框中内联图标的颜色样式。目前内置提供16种样式：dark，red，light-red，blue，light-blue，green，light-green，orange，light-orange，orange2，purple，pink，pink2，brown，grey，light-grey。 String blue
Value 文本框的文本内容。 String null
ReadOnly 设置一个值，该值标识文本框是否为只读模式。 Boolean false
Disabled 设置一个值，该值标识文本框是否禁用。 Boolean false
CanMultiLine 设置一个值，该值标识文本框是否为多行文本框，即 textarea。 Boolean false
AutoSize 当文本框为多行文本框，该属性设置文本框高度是否随内容高度自动伸缩。 Boolean false
Width 文本框的宽度。 Number null
Height 文本框的高度。注意，此属性只有当文本框为多行文本框时才生效。 Number null
Options 事件属性对象详细说明见下表：
名称 参数 具体描述
OnBeforeFocus sender 在触发 Focus 焦点事件前触发，可以通过 return false 来阻止继续操作。
OnFocus sender 当设置文本框焦点时触发。
OnBlur sender 当焦点离开文本框上时触发。
OnChange value 当文本框中的值发生变化时触发。value 为最新文本值。
TextBox 对象方法说明见下表：
调用：$("id").Frame().方法名();
方法 参数 具体描述
Enable none 启用文本框。
Disable none 禁用文本框。
Focus none 将焦点设置在文本框上。
Readable none 设置文本框为可读写。
UnReadable none 设置文本框为只读。

html代码：

<input id="txtDefault" class="frame-textbox" type="text" value="默认样式文本框" />
<input id="txtWithIconOnRight" class="frame-textbox" type="text" value="右侧带图标的文本框" />
<input id="txtWithIconOnLeft" class="frame-textbox" type="text" value="左侧带图标的文本框" />                    
                    
<input id="txtMini" class="frame-textbox" type="text" value="mini 尺寸的文本框" />
<input id="txtSamll" class="frame-textbox" type="text" value="small 尺寸的文本框" />
<input id="txtMedium" class="frame-textbox" type="text" value="medium 尺寸的文本框" />
<input id="txtLarge" class="frame-textbox" type="text" value="large 尺寸的文本框" />
<input id="txtXLarge" class="frame-textbox" type="text" value="xlarge 尺寸的文本框" />
<input id="txtXXLarge" class="frame-textbox" type="text" value="xxlarge 尺寸的文本框" />                    
                    
<textarea id="txtMultiRow" class="frame-textbox" rows="9" cols="9" style="height:69px;">多行文本</textarea>
<textarea id="txtAutoSize" class="frame-textbox autosize-transition" rows="9" cols="9" style="height:69px;">随内容自动变化高度</textarea>

javascript代码：

$("#txtDefault").TextBox();
$("#txtWithIconOnRight").TextBox({ Icon: "icon-leaf", IconSite: "right", Width: 160 });
$("#txtWithIconOnLeft").TextBox({ Icon: "icon-leaf", Width: 160 });

$("#txtMini").TextBox({ Size: "mini", Width: 100 });
$("#txtSamll").TextBox({ Size: "small", Width: 110 });
$("#txtMedium").TextBox({ Size: "medium" });
$("#txtLarge").TextBox({ Size: "large", Width: 200 });
$("#txtXLarge").TextBox({ Size: "xlarge", Width: 200 });
$("#txtXXLarge").TextBox({ Size: "xxlarge", Width: 200 });

$("#txtMultiRow").TextBox({ CanMultiLine: true });
$("#txtAutoSize").TextBox({ CanMultiLine: true, AutoSize: true });
