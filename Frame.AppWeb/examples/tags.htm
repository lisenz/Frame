Tags

Tags 是一个标签列表容器控件，通过使用 ul 列表添加 .frame-tags 样式。

通过 Frame 组件形式生成 Tags 标签条，编写 var tags = $("#页面标签id").Tags(options); 这样一行代码，即可将页面指定ID的标签通过 options 相关属性的配置生成相应的标签容器。
Options 属性对象详细说明见下表：
属性名 具体描述 类型 默认值
Items 标签对象集合。 Array null
Plain 设置一个值，该值标识是否为简洁模式，即无边框。 Boolean true
Width 标签容器的宽度。 Number null
Options 事件属性对象详细说明见下表：
名称 参数 具体描述
OnItemClick sender，index 点击标签时触发。触发时传递当前标签元素和其在容器中的索引位置。
Tags 对象方法说明见下表：
调用：$("id").Frame().方法名();
方法 参数 具体描述
AppendTag tag 往标签列表中追加一个标签对象。tag 对象格式为{ Text，Closable }，其中 Text 为标签文本内容，Closable 标识该标签是否带关闭功能按钮。
AppendRangeTag tags 往标签列表中批量追加多个标签对象。tags 为一个数组对象。
RemoveTag which 从标签列表中移除指定文本或指定索引的标签，which 标识标签的文本或索引值。
Exists which 判断指定文本或指定索引的标签是否存在于标签列表中，返回 true 表示已存在，否则为不存在。
GetTagIndex text 获取指定名称的标签在标签列表中对应的索引值。

html代码：

<ul id="taglist"></ul>                    
<ul id="taglistWithBorder"></ul>

javascript代码：

$("#taglist").Tags({ Items: [
    { Text: "Tag1", Closable: true },
    { Text: "Tag2", Closable: true },
    { Text: "Tag3", Closable: true },
    { Text: "Tag4", Closable: true }
    ],
    OnItemClick: function (sender, index) { $.Messager.Alert("系统消息", $(sender).text(), "info"); }
});

$("#taglistWithBorder").Tags({ Plain: false, Items: [
        { Text: "Tag1", Closable: true },
        { Text: "Tag2", Closable: true },
        { Text: "Tag3", Closable: true },
        { Text: "Tag4", Closable: true }
    ],
    OnItemClick: function (sender, index) { $.Messager.Alert("系统消息", $(sender).text(), "info"); }
});
