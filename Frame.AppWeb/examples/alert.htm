Alert

Alert 是一个消息提醒控件，可以嵌入到页面的任何位置。通过在 div 标签添加 .frame-alert 标识为 Frame 消息提醒控件。
继而使用 var alert = $("#页面标签id").Alert(options); 这样一行代码，根据 options 相关属性的配置将目标元素生成为消息提醒控件对象。

Options 属性对象详细说明见下表：
属性名 具体描述 类型 默认值
Width 消息提醒控件的宽度。 Number null
Icon 设置消息提醒的图标样式名称。 String null
Size 消息提醒控件的尺寸，默认为标准尺寸。有两种尺寸可选：large，small。 String null
Content 消息提醒的内容。内容的类型可为 HTML 片段、文本或使用回调函数返回的值。当目标元素已包含有子元素内容，则该属性失效。 String null
Mode 消息提醒的背景色模式，有4种模式可选：info，success，warning，error。 String success
Closable 设置一个值，该值标识消息提醒创建之后是否为关闭模式。 Boolean false
Alert 对象方法说明见下表：
调用：$("id").Frame().方法名();
方法 参数 具体描述
Toggle none 切换显示或关闭消息提醒控件。
IsVisible none 返回一个值，该值标识消息提醒控件是否可见。
SetContent content 设置消息提醒内容。
Show none 显示消息提醒控件。
Close none 取消显示消息提醒控件。

html代码：

<div id="alert">默认消息提醒</div>
<div id="alertHasIcon">带图标的消息提醒</div>                                        
<div id="alertOnSuccess">success 模式的消息提醒</div>
<div id="alertOnError">error 模式的消息提醒</div>
<div id="alertOnWarning">warning 的消息提醒</div>
<div id="alertOnInfo">info 模式的消息提醒</div>
<div id="alertForLarge">大尺寸的消息提醒</div>
<div id="laertForSmall">小尺寸的消息提醒</div>

javascript代码：

$("#alert").Alert();
$("#alertHasIcon").Alert({ Icon: "icon-question-sign" });

$("#alertOnSuccess").Alert({ Icon: "icon-question-sign", Mode: "success" });
$("#alertOnError").Alert({ Icon: "icon-question-sign", Mode: "error" });
$("#alertOnWarning").Alert({ Icon: "icon-question-sign", Mode: "warning" });
$("#alertOnInfo").Alert({ Icon: "icon-question-sign", Mode: "info" });

$("#alertForLarge").Alert({ Icon: "icon-question-sign", Size: "large" });
$("#laertForSmall").Alert({ Icon: "icon-question-sign", Size: "small" });
