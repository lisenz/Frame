Radio

Radio 用于创建单选框。它是将正常的 type = "radio" 的 input 元素转换为 Radio 控件对象。
通过 Frame 组件形式生成复选框控件，使用 var radio = $("#页面标签id").Radio(options); 实现将目标元素转换为 Radio 控件对象。

Options 属性对象详细说明见下表：
属性名 具体描述 类型 默认值
Value 显示在单选框上的文本。 String null
Checked 设置一个值，该值标识单选框是否勾选中。 Boolean null
Disabled 设置一个值，该值标识是否禁用单选框。 Boolean false
Options 事件属性对象详细说明见下表：
名称 参数 具体描述
OnChange checked 当单选框的勾选状态发生改变时触发。
Radio 对象方法说明见下表：
调用：$("id").Frame().方法名();
方法 参数 具体描述
Enable none 启用单选框。
Disable none 禁用单选框。
Checked checked 对单选框进行勾选或取消勾选。当 checked 为 true 时，单选框被勾选；否则为取消勾选。
Status none 获取当前单选框的勾选状态。

html代码：

<input id="Radio3" type="radio" />
<input id="Radio4" class="frame-radio" checked="checked" value="男" type="radio" name="frame" />
<input id="Radio5" class="frame-radio" type="radio" name="frame" />

javascript代码：

$("#rd").Radio();
$("#Radio1").Radio();
$("#Radio2").Radio({ Value: "女" });
