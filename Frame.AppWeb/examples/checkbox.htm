CheckBox

CheckBox 用于创建复选框。它是将正常的 type = "checkbox" 的 input 元素转换为 CheckBox 控件对象。
通过 Frame 组件形式生成复选框控件，使用 var chk = $("#页面标签id").CheckBox(options); 实现将目标元素转换为 CheckBox 控件对象。

Options 属性对象详细说明见下表：
属性名 具体描述 类型 默认值
Value 显示在复选框上的文本。 String null
Checked 设置一个值，该值标识复选框是否勾选中。 Boolean null
Disabled 设置一个值，该值标识是否禁用复选框。 Boolean false
Options 事件属性对象详细说明见下表：
名称 参数 具体描述
OnChange checked 当复选框的勾选状态发生改变时触发。
CheckBox 对象方法说明见下表： 调用：$("id").Frame().方法名();
方法 参数 具体描述
Enable none 启用复选框。
Disable none 禁用复选框。
Checked checked 对复选框进行勾选或取消勾选。当 checked 为 true 时，复选框被勾选；否则为取消勾选。
Toggle none 对复选框进行勾选或取消勾选，自动切换复选框的勾选状态。

html代码：

<input id="chk" type="checkbox" value="复选框" />
<input id="chkOnDisable" type="checkbox" value="(禁用)复选框" />
<input id="chkOnChecked" type="checkbox" value="(勾选)复选框" />

javascript代码：

$("#chk").CheckBox();
$("#chkOnDisable").CheckBox({ Disabled: true });
$("#chkOnChecked").CheckBox({ Checked: true });
