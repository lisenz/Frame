Panel

Panel 可作为其他内容的容器，可以嵌入到页面的任何位置。通过在 div 标签添加 .frame-panel 标识为 Frame 面板控件容器。
通过使用 var panel = $("#页面标签id").Panel(options); 这样一行代码，即可将目标元素通过 options 相关属性的配置生成相应的面板控件。

Options 属性对象详细说明见下表：
属性名 具体描述 类型 默认值
HasHeader 标识一个值，该值标识面板是否使用标题栏。 Boolean false
Title 面板标题栏的标题文本。 String 无标题
Icon 面板标题栏的图标样式名称。 String icon-desktop
ToolBar 嵌入面板标题栏的拓展工具栏。项对象结构：{Icon，OnClick} Array null
HasFooter 设置一个值，该值标识是否使用命令栏。 Boolean false
Commands 嵌入面板命令栏的命令按钮控件集合对象。集合的项对象为 Frame 组件的 Button 控件对象。 Array null
Width 面板宽度。 Number null
Height 面板高度。 Number null
Plain 设置一个值，该值标识是否为简洁模式,即无边框。 Boolean false
Url 加载远程页面的URL，然后显示在面板主体内容中。 String null
Params 指定给加载远程页面的URL的参数对象。 Object null
Content 面板的主体内容。 String null
面板的 ToolBar 项对象的属性详细说明见下表：
属性名 具体描述 类型
Icon 工具项的图标样式名称。 String
OnClick 工具项的单击事件。 Function
Panel 对象方法说明见下表：
调用：$("id").Frame().方法名();
方法 参数 具体描述
Load url，params 远程加载一个新页面的Url。
Reload params 重新远程刷新加载当前页面。

html代码：

<div id="panel1" class="frame-panel">
    <div>容器内容已嵌入</div>
</div>
<div id="panel2" class="frame-panel"></div>
<div id="panel3" class="frame-panel"></div>

javascript代码：

$("#panel1").Panel({
    HasHeader: true,
    Title: "标题",
    ToolBar: [
        {
            Icon: "icon-refresh",
            OnClick: function (context) {
            }
        },
        {
            Icon: "icon-chevron-up",
            OnClick: function (context) {
            }
        }
    ],
    HasFooter: true,
    Commands: [
        { Icon: "icon-arrow-right", IconSite: "left", Text: "下一页", Orientation: "right" },
        { Icon: "icon-arrow-left", IconSite: "left", Text: "上一页" }
    ]
});
