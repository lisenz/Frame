Button Groups

Button Group 作为静态组件使用，用于将一组按钮编制为按钮组。通过在 div 标签上添加 .frame-button-groups 将目标元素标识为按钮组容器 。
用户可以在按钮组中放置任意数量的 button 元素。

详细用法可以参考 代码 标签页的示例代码。

html代码：

<div class="frame-button-groups">
    <button class="frame-button"><i class="icon-plus"></i><span>新增</span></button>
    <button class="frame-button"><i class="icon-edit"></i><span>修改</span></button>
    <button class="frame-button"><i class="icon-trash"></i><span>删除</span></button>
    <button class="frame-button"><i class="icon-search"></i><span>查询</span></button>
</div>
<div class="frame-button-groups">
    <button id="btn1">新增</button>
    <button id="btn2">修改</button>
    <button id="btn3">删除</button>
    <button id="btn4">查询</button>
</div>

javascript代码：

$("#btn1").Button({ Icon: "icon-plus" });
$("#btn2").Button({ Icon: "icon-edit" });
$("#btn3").Button({ Icon: "icon-trash" });
$("#btn4").Button({ Icon: "icon-search" });
